# CAP Sleep Database v1.0.0

Source: PhysioNet
Published: 2012-07-26
DOI: 10.13026/C2VC79
License: Open Data Commons Attribution License v1.0

## Key facts
- Open dataset with 108 polysomnographic recordings from the Sleep Disorders Center of the Ospedale Maggiore of Parma, Italy.
- Includes 16 healthy controls and 92 pathological recordings.
- Signal content includes at least 3 EEG channels, EOG, ECG, respiration channels, and EMG channels.
- PhysioNet `RECORDS` file includes `brux1.edf` and `brux2.edf`, making an open bruxism subset directly discoverable.
- This is the simplest openly available starting point found for bruxism EEG/EMG experimentation.

## Extracted notes
The CAP Sleep Database is primarily a sleep-instability / CAP resource, not a purpose-built bruxism benchmark. That matters: the public bruxism subset is tiny, so any starter project should be framed as a reproducible pilot / reproduction exercise rather than a clinically meaningful detector.
