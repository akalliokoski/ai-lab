# Number of masseteric electromyographic waveforms during analysis periods with/without excluding time zones after going to bed and before getting up in sleep bruxism assessment

Source: PubMed / Cranio (2026)
PMID: 40062531

## Abstract-style summary
- Measured masseter EMG in 22 outpatients with sleep bruxism.
- Used a wearable motion monitor to estimate sleep onset and wake-up times, then compared several analysis windows around time in bed versus likely sleep.
- Episode counts per hour were highly correlated across windows, but counting the full time in bed inflated event rates relative to sleep-onset-to-wake periods.
- Excluding plausible wakefulness near bedtime and final awakening improved agreement without requiring full PSG.

## Takeaway for ai-lab
This is a useful translational reminder that event counts depend on how the analysis window is defined. It supports keeping the repo explicit about selection windows and discourages over-reading crude episode-rate comparisons from different scaffolds.
